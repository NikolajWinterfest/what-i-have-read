### Цель проекта:

Создать локальный сайт для управления списком прочитанных книг, где можно добавлять книги в избранное, создавать, изменять и удалять карточки книг, а также добавлять цитаты

### Задачи проекта:

#### Создание списка книг:

– Отображение всех добавленных книг в виде карточек<br />
– Создание / изменение / удаление карточек книг и информации о них<br />
– Добавление книг в избранное<br />

#### Управление книгами:

– Создание новой карточки книги с полями: название, автор, описание, обложка, перевод<br />
– Редактирование существующих книг<br />
– Удаление книг с подтверждением<br />

#### Управление цитатами:

– Добавление цитат к книгам<br />
– Редактирование и удаление цитат<br />

#### Загрузка файлов:

– Загрузка обложек книг<br />
– Хранение файлов на сервере<br />

#### Серверная часть:

– REST API для управления данными книг<br />
– Хранение данных в JSON файле<br />
– Обработка загрузки файлов<br />

## Запуск проекта

### Склонировать проект на ваш компьютер с Github с помощью команды:

```
git clone NikolajWinterfest/what-i-have-read
```

### Установить зависимости с помощью команды

```
npm install
```

### Запустить проект с помощью команды

```
npm run dev
```

### Собрать проект с помощью команды

```
npm run build
```
